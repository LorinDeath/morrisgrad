# MorrisGrad

Добро пожаловать в **МоррисГрад** — цифровое воплощение нашего удивительного мира.

Этот проект создан на базе [Astro](https://astro.build), чтобы рассказать историю города, показать его вселенную и передать уникальную атмосферу энергии и стиля.

МоррисГрад — это место, где технологии встречаются с магией. Здесь вы найдете:

- Интересный сюжет мира
